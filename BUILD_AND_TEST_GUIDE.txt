QUICK START:
-----------
1. Extract all files to a single directory
2. Run: make all
3. Run: ./cellular_network

DETAILED INSTRUCTIONS:
---------------------

STEP 1: Verify File Structure
-----------------------------
Ensure you have all 8 files in the same directory:

Required files:
- CellularNetwork.h
- CellularNetwork.cpp
- main.cpp
- basicIO.h
- basicIO.cpp
- syscall.s
- Makefile
- README.txt

Verify with: ls -la

STEP 2: Build the Project
-------------------------

Option A - Build Everything (Recommended):
$ make all

This creates two binaries:
- cellular_network_debug   (with debugging symbols, -g -O0)
- cellular_network         (optimized, -O3)

Option B - Build Only Debug Version:
$ make debug

Option C - Build Only Release Version:
$ make release

STEP 3: Run the Simulator
-------------------------

Run the optimized version:
$ ./cellular_network

OR

Run the debug version (useful for troubleshooting):
$ ./cellular_network_debug

OR

Use make targets:
$ make run          # Builds and runs release version
$ make run-debug    # Builds and runs debug version

TROUBLESHOOTING:
---------------

Problem 1: Compilation Errors
-----------------------------
Error: "g++: command not found"
Solution: Install g++
  Ubuntu/Debian: sudo apt-get install g++
  Fedora/RHEL: sudo yum install gcc-c++
  Arch: sudo pacman -S gcc

Error: "unsupported C++17"
Solution: Ensure g++ version >= 7.0
  Check: g++ --version
  Upgrade if necessary

Problem 2: Linker Errors
------------------------
Error: "undefined reference to syscall3"
Solution: Ensure syscall.s is being compiled and linked
  Check Makefile is correct
  Verify architecture is x86-64

Problem 3: Runtime Errors
-------------------------
Error: Segmentation fault
Solution: 
  1. Run the debug version: ./cellular_network_debug
  2. Use gdb: gdb ./cellular_network_debug
  3. Check that basicIO.cpp is correctly implemented

Problem 4: Make Errors
---------------------
Error: "make: *** No targets specified"
Solution: 
  1. Ensure Makefile has no tabs converted to spaces
  2. Verify Makefile is named exactly "Makefile"
  3. Try: make -f Makefile all

COMPILER FLAGS EXPLAINED:
------------------------

Debug Build (-g -O0 -DDEBUG):
  -g        : Include debugging symbols
  -O0       : No optimization (easier to debug)
  -DDEBUG   : Define DEBUG macro

Release Build (-O3 -DNDEBUG):
  -O3       : Maximum optimization
  -DNDEBUG  : Disable assertions

Common Flags (-std=c++17 -Wall -Wextra):
  -std=c++17 : Use C++17 standard
  -Wall      : Enable all warnings
  -Wextra    : Enable extra warnings

PERFORMANCE COMPARISON:
